# Projects and Exercises

This section contains exercises to build practical ethical hacking skills.

## Capture the Flag (CTF) Challenges
- **TryHackMe**: Solve challenges on networking and web vulnerabilities.
- **Hack The Box**: Complete at least one beginner-level CTF.

## Automated Scripts
- **Network Reconnaissance Script**: Write a Python/Bash script to automate network scanning.

## Vulnerability Assessment
- Choose a vulnerable web app (e.g., DVWA) and create a report detailing your findings and recommendations.
